# VsCode

主要记录插件、配置等

## 插件

### 1.【主题、Icon】Catppuccin

Catppuccin是一种由社区驱动的粉彩主题，旨在介于低对比度与高对比度主题之间。

-   [Github](https://github.com/catppuccin/catppuccin)

### 2.【增强】Errorlens

ErrorLens 通过让错误突出显示，增强了语言诊断功能的提升。

-   [Github](https://github.com/usernamehw/vscode-error-lens)

### 3.【增强】Jetbrains IDE Keymap

该扩展将 JetBrains 的按键绑定导入 VS Code。安装扩展并重启 VS Code 后，你可以像 IntelliJ IDEA、Webstorm、PyCharm 等一样使用 VS Code。

-   [Github](https://github.com/isudox/vscode-jetbrains-keybindings)

### 4.【服务】Live Server

启动一个本地开发服务器，支持静态和动态页面的实时加载功能。

-   [Github](https://github.com/ritwickdey/vscode-live-server)
