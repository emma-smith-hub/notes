# 存储桶

存储桶、图床等软件

## 1. PicGo

无缝将图片上传融入您的创意工作流程，可接入 `Typora`、`Obsidian` 等自动上传

-   [官网](https://picgo.app/)
-   [Github](https://github.com/Molunerfinn/PicGo)

## 2. PicList

一站式全能云存储与图床管理工具

-   [官网](https://piclist.cn/)
-   [Github](https://github.com/Kuingsmile/PicList)

## 3. WinSCP

在本地计算机与远程服务器之间复制文件，使用FTP、FTPS、SCP、SFTP、WebDAV 或 S3 文件传输协议.

-   [官网](https://winscp.net/eng/index.php)
