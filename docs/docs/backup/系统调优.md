# 系统调优

电脑系统的一些优化

## 风扇

### CPU 风扇

| 温度 | 转速 |
| ---- | ---- |
| 40°C | 30%  |
| 65°C | 50%  |
| 80°C | 75%  |
| 90°C | 100% |

>   [!WARNING] 提示
>
>   AMD Ryzen 9 7950X
>
>   设计温度是 **95°C**。

| 75°C | 85%  |

### 后置风扇

| 温度 | 转速 |
| ---- | ---- |
| 40°C | 30%  |
| 60°C | 45%  |
| 75°C | 65%  |
| 85°C | 85%  |

>   [!WARNING] 提示
>

up time -> 3s
