# 系统软件

增强系统软件

## 1. Auto Dark Mode

在定时时间在 Windows 的暗色和浅色主题之间切换。

-   [Github](https://github.com/AutoDarkMode/Windows-Auto-Night-Mode)
-   [Microsoft Store](https://apps.microsoft.com/detail/xp8jk4hzbvf435?hl=zh-CN&gl=CN)



## 2. LiteMonitor

免费 · 开源 · 轻量 · 桌面硬件性能监控软件，实时监控 CPU / GPU / 显存 / 内存 / 磁盘 / 网络

-   [官网](https://litemonitor.cn/)
-   [Github](https://github.com/Diorser/LiteMonitor)



## 3. Traffic Monitor

Traffic Monitor是一款用于Windows平台的网速监控悬浮窗软件，可以显示当前网速、CPU及内存利用率，支持嵌入到任务栏显示，支持更换皮肤、历史流量统计等功能。

-   [Github](https://github.com/zhongyang219/TrafficMonitor?tab=readme-ov-file)
