# 视频处理

一些视频处理的开源工具

## 1. Lossless Cut

跨平台 FFmpeg 图形界面，实现视频、音频、字幕及其他相关媒体文件的极快速且无损操作。

-   [Github](https://github.com/mifi/lossless-cut)

## 2. Smart Cut

`smartcut`可以以多种格式和编解码器切割视频文件，且在剪切点周围只需最小的重编码（即“智能剪辑”）。

**该开源软件不再更新（可继续使用 Github 下载即可），付费版在官网**

-   [Github](https://github.com/skeskinen/smartcut)
-   [官网](https://smartmediacutter.com/zh-CN/pricing/)

## 3. 开源视频工具全景图

AI视频生成模型、在线编辑器、短视频自动化 `Pipeline`、视频理解分析工具。

-   [官网](https://doc-openvideo.vercel.app/)
